# KNN Classification Project

This project applies the K-Nearest Neighbors (KNN) algorithm to classify a dataset.  
The data was preprocessed using feature scaling to ensure accurate distance calculations.

Multiple values of K were tested, and the optimal value (K=33) was selected based on the lowest error rate from the graph.  
The model was then retrained using this value, resulting in improved classification performance.

The evaluation was conducted using a confusion matrix and classification report.
